Classroom Management - ServiceNow/Accenture Hackathon

My Role

Application Development

We all have to face itâ€¦2020 has be a very tough year for everyone. The pandemic has brought so many issues that no one planned for including social distancing, which resulted in the necessary decrease in the number of people indoors.

Class Management tries to resolve the problem concerning the essential decrease to the number of students attending in-person courses. Using the ServiceNow platform, the application allows students to see the courses they are enrolled in and send a request if they would prefer to take the class in person or online.

The design of the application consists of creating forms for student requests and the creation of 5 main tables (Students, Instructors, Courses, Facilities, and Course Enrollment).

WORKFLOW

The tables use relationships for communication, allowing an organized flow of information between the tables. For example, a student may be enrolled into different courses, and in the course enrollment table, there would be a reference to the student table and the course table. In the course table, there would then be a reference to the facility table to obtain the maximum number of people that can fit in a classroom. The number of remaining seats for in-person class attendance will be shown in the Class Enrollment table, which can be seen by students when trying to request their preference.

FUTURE IMPLEMENTATION

Additional future implementation of the application would allow much improvement to the application such as notification to students via email and sms, which would notify them if there are any changes in the remaining seats for in-person attendance. Further enhancement would include the addition of time schedules for cleaning service to provide security and safety for both students and staff.
